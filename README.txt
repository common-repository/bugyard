== Description ==

This plugins help you quickly deploy bugyard feedback widget in your wordpress website.

1. Enabling user to send detailed feedback directly from within the app without any hassle
2. Getting access to the feedback meta data without asking the user anything
3. Get access to a screenshot of the website when the user made his feedback
4. Get access to the session information, cookies and computer information
5. Follow up with the feedback directly from the website interface.


== Installation ==

This section describes how to install the plugin and get it working.

1. Download the plugin and copy it in the following directory of your wordpress website /wp-content/plugins
2. Activate the plugin through the 'Plugins' menu in WordPress
3. In Setting > Bugyard Thick "Activate Bugyard on your Wordpress Website" and copy your ID in the field below
4. You are all set
